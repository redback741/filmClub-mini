基础说明
- 鉴权方式：Bearer JWT（请求头 Authorization: Bearer <token>）
- 基础请求封装：见 src/utils/request.js（自动附加 token、统一拦截响应、支持 GET/POST/DELETE 等方法与 query/body 组合）
- 环境变量：服务端基础地址来自 process.env.TARO_APP_BASE_URL

用户模块（src/api/user.js）
- 登录
  - 方法：wxLogin(params)
  - 请求：POST /user/wx-login
  - 入参：{ code, userInfo?, encryptedData?, iv? }
- 更新用户信息
  - 方法：updateUserInfo(params)
  - 请求：POST /user/update
  - 入参：{ nickName?, avatar?, password? }
- 查询用户信息
  - 方法：getUserInfo()
  - 请求：GET /user/info
- 删除用户
  - 方法：deleteUser()
  - 请求：DELETE /user/delete

活动模块（src/api/activity.js）
- 创建活动
  - 方法：createActivity(params)
  - 请求：POST /activity/create
  - 入参：CreateActivityDto（包含 name、startTime、screeningTime、type、movieName、city、address、hallType、recruiterName、price 等）
- 查询活动列表
  - 方法：listActivities(params?)
  - 请求：GET /activity/list
  - 入参：可选查询参数
- 删除活动
  - 方法：deleteActivity(id)
  - 请求：DELETE /activity/delete?id=number
  - 入参：id（query）
- 更新活动
  - 方法：updateActivity(id, params)
  - 请求：POST /activity/update?id=number
  - 入参：id（query）+ CreateActivityDto（body）
- 更新活动状态
  - 方法：updateActivityStatus(id, status)
  - 请求：POST /activity/update-status?id=number&status=string
  - 入参：id、status（query）

  name: '《沙丘2》IMAX首映礼',
  startTime: '2024-03-08T14:00:00Z',
  screeningTime: '2024-03-08T15:00:00Z',

  posterUrl: 'https://example.com/poster.jpg',
  city: '北京',
  address: '朝阳区建国路93号万达广场',
  hallType: 2,
  recruiterName: 'IMAX Club',
  price: 0
})

// 活动列表
const listRes = await listActivities()

// 更新活动

// 更新状态

// 删除活动
