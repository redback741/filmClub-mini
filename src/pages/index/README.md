# 首页模块文档

## 模块功能

- 模块路径：`src/pages/index`
- 页面用途：项目首页入口页，用于展示首页标题和功能卡片入口。
- 当前交互：
  - 点击“浏览电影”跳转到 `pages/home/index`
  - 点击“我的报名”跳转到 `pages/myActivity/index`
  - 点击“活动列表”跳转到 `pages/list/index`

## 文件说明

- `index.jsx`：页面结构与跳转逻辑
- `index.scss`：页面视觉样式
- `index.config.js`：页面配置
